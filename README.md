## Netflix 홈페이지
+ HTML, CSS, JS를 공부하며 만든 넷플릭스 홈페이지로 작지만 이 프로젝트를 통해 홈페이지의 레이아웃 구조에 대해 더 공부 할 수 있었으며,
  기존 홈페이지에 js를 활용하여 조금 더 동적인 효과를 주었습니다.
